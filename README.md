## üöÄ Getting Started

### Prerequisites
1. Install [Node.js](https://nodejs.org/) (v14+)
2. Install Jupyter: `pip install jupyter`
3. Install IJavascript kernel:
   ```bash
   npm install -g ijavascript
   ijsinstall
   ```

### Running the Notebooks
```bash
jupyter notebook
```
Then open any `.ipynb` file and start learning!

## üìä Complexity Reference

| Category | Best For | Time | Space |
|----------|----------|------|-------|
| Hash Table | Lookup | O(1) | O(n) |
| Binary Search | Sorted arrays | O(log n) | O(1) |
| Merge Sort | Stable sorting | O(n log n) | O(n) |
| Quick Sort | General sorting | O(n log n) avg | O(log n) |
| BFS/DFS | Graph traversal | O(V + E) | O(V) |
| DP | Overlapping subproblems | Varies | Varies |
